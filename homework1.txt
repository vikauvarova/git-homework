        Создать учетную запись на гитхаб
        Создать репозиторий
        В терминале получить SSH ключ
        В настройках добавить этот ключик
        git config --global user.name "Viktoriia Uvarova"
        git config --global user.email "vg220915@gmail.com"
        Зайти в рабочую директорию в которую хотим скопировать
        В терминале написать команду git clone ссылка на репозиторий

2. Как создать git репозиторий из локального каталога на своем комьютере. Как его отправить на github.com?

        echo "# homework1" >> README.md
        git init
        git add README.md
        git commit -m "first commit"
        git branch -M main
        git remote add origin https://github.com/vikauvarova/homework1.git
        git push -u origin main

4. Какую информацию нам дают команда :
git status  ---> текущее состояние нашего гит репозитория
git log -----> перечисляет коммиты, сделанные в репозитории в обратном к хронологическому порядке — последние коммиты находятся вверху.
git remote -v  ---> просмотреть, какие удалённые серверы уже настроены, ключик -v показивает какому URL соответствует сокращённое имя в Git
